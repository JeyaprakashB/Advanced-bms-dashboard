🔋 Advanced Battery Management System Dashboard

 📌 Description
This project simulates an advanced Battery Management System (BMS) with thermal monitoring.

 ⚡ Features
- State of Charge (SOC) calculation
- Charging / Discharging detection
- Temperature monitoring
- Safety alerts (Overheating, Overcurrent)
- Battery level progress bar
- Modern dashboard UI (CustomTkinter)

 🛠 Technologies Used
- Python
- CustomTkinter

 ▶ How to Run
1. Install Python
2. Install dependencies:
   pip install customtkinter
3. Run:
   python main.py

 🎯 Applications
- Electric Vehicles (EV)
- Battery Packs
- Energy Storage Systems
